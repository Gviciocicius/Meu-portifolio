Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir um pouco sobre minha vida estudatil. O objetivo é compartilhar algumas informações como uma espécie de cartão de visitas, fornecendo uma descrição breve da minha vida estudantil.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária..
